# 인구감소지역 관계인구 활성화를 위한 코사인 유사도 기반 인플루언서 매칭 시스템

##  프로젝트 개요
본 프로젝트는 인구감소지역의 **관계인구 활성화**를 목적으로,  
관광지 데이터와 인플루언서 데이터를 분석하여  
지역 특성에 가장 적합한 인플루언서를 매칭하는 시스템이다.

지역과 인플루언서 간의 의미적 유사도를 반영하기 위해  
**토픽 모델링, 키워드 분석, 단어 임베딩, 코사인 유사도**를 활용하였다.

---

##  시스템 파이프라인

### 1. 관광지 데이터 분석 (Region Side)
- 관광지 관련 텍스트 데이터를 기반으로 **LDA 토픽 모델링** 수행
- 관광자원 중심의 **지역별 핵심 키워드** 도출
- 도출된 키워드를 **FastText**를 이용해 단어 임베딩 벡터로 변환

---

### 2. 인플루언서 데이터 분석 (Influencer Side)
- 인플루언서 콘텐츠(해시태그, 설명 등)에 대해 **TF-IDF** 적용
- 지역별 인플루언서의 **방문 목적 및 콘텐츠 성향 키워드** 추출
- 추출된 키워드를 **FastText** 기반 단어 임베딩 벡터로 변환

---

### 3. 유사도 계산 및 매칭
- 지역 키워드 벡터와 인플루언서 키워드 벡터 간의  
  **Cosine Similarity** 계산
- 유사도가 높은 인플루언서를  
  해당 지역에 **가장 어울리는 인플루언서로 매칭**

---

##  기대 효과
- 지역 특성과 인플루언서 콘텐츠 간의 정합성 향상
- 일회성 방문이 아닌 **지속적인 관계인구 형성** 유도
- 데이터 기반 지역 홍보 및 마케팅 전략 수립 가능

---

##  사용 기법 요약
- Topic Modeling: LDA  
- Keyword Extraction: TF-IDF  
- Word Embedding: FastText  
- Similarity Measure: Cosine Similarity
